# shopIT_Ecommerce_project
ShopIT is a modern, feature-rich eCommerce application built with the MERN Stack (MongoDB, Express.js, React, Node.js). It provides a seamless shopping experience for users and a robust management dashboard for administrators.

For Users
Authentication & Profiles: Secure login/signup with JWT and profile management (avatar uploads via Cloudinary).

Product Discovery: Advanced search, category filtering, and price range sliders.

Shopping Cart: Persistent cart functionality with real-time price updates.

Checkout & Payments: Secure payment integration using Stripe.

Order History: Track order status and view past purchases.


For Admins
Admin Dashboard: Overview of sales, products, and user statistics.

Product Management: Full CRUD (Create, Read, Update, Delete) functionality for products.

Order Management: Process and update order delivery statuses.

User Management: Manage user roles and permissions.

Tech Stack
Frontend: React.js, Redux Toolkit (State Management), Bootstrap.

Backend: Node.js, Express.js.

Database: MongoDB Atlas (NoSQL).

Storage: Cloudinary (for product and profile images).

Authentication: JWT (JSON Web Tokens) & Bcrypt.js.

Payments: Stripe API.
